# PJCS Training Lab v0.6.1

Pokémon GO スーパーリーグ向けの6→3選出・自動対戦シミュレーターです。

## v0.6の主な修正

- 「勝てる選出」の％を透明化
  - 相手の60選出をすべて同じ重みで評価
  - 上位候補は各選出につき60選出 × 3乱数 = 180試合
  - 勝数・敗数・総試合数をカード内に表示
  - 実際の大会使用率による重み付けは行っていないことを明記
- 選出解説を具体化
  - 選出した各ポケモンが、相手の誰に刺さるかを個別表示
  - この3体では回答できない「重い相手」を表示
  - 回答が1体に集中する相手、先発が0勝3敗になる相手を警告
  - 相手6体それぞれへの最善回答と0・1・2枚同数シールド結果を表示
- 6体登録画面
  - 「変更」ボタンを必ず横書きに固定
  - A / D を廃止し、「攻撃」「防御」「HP」と日本語で表示
  - ポケモン名の横にスプライト画像を表示
- 選出画面・検索画面・データ一覧にも画像を追加
- v0.5以前の古いおすすめ結果を再表示しないようキャッシュを更新

## 勝率の意味

「勝てる選出」に出る％は、現実の大会勝率や使用率を予測した数字ではありません。相手の6体から作れる60通りを等確率と仮定し、シミュレーター内AIで自動対戦した結果です。

## ポケモン画像

画像ファイルはZIP内へ同梱していません。PokéAPIの公開スプライトリポジトリから外部読み込みし、通信できない場合は文字アイコンへ切り替わります。Pokémonおよび各画像の権利は各権利者に帰属します。


- v0.6.1: フォルム違いポケモンの画像表示を修正。原種とリージョンフォームが同じ写真にならないよう、対応済みフォームは個別スプライトを使用し、未検証の特別フォームは誤表示を避けるため文字アイコンへフォールバックします。
