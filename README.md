# PJCS Pokémon GO Simulator v0.3

iPhoneのSafariからホーム画面に追加して使用できるPWAです。

## v0.3の主な変更

- PvPokeの現行オープン・スーパーリーグランキングとゲームマスターデータをオンライン取得
- 有効な上位250体を端末へキャッシュ（最低200体を検証してから採用）
- ポケモン検索・6体登録UI
- エンペルトの通常技名を「きんぞくおん」へ修正
- 6体から考えられる60通りの選出を分析
- 相手の全60選出候補を想定して、おすすめ3選出と理由を表示
- 1試合自動対戦、100試合勝率分析、BO3/BO5進行

## 初回起動

初回だけインターネット接続が必要です。「データ更新」でPvPokeのデータを読み込みます。読み込み後は処理済みデータを端末に保存します。

## 注意

PvPokeランキングはシミュレーションに基づく環境指標であり、Niantic公式の使用率ではありません。おすすめ選出と勝率も、本アプリのAIモデルによる推定値です。
