# PJCS Pokémon GO Simulator v0.2 beta

iPhoneのSafariで動くPWAです。

## v0.2の機能
- 6体登録、3体選出、BO3/BO5
- 3対3の自動シミュレーション
- 通常技のターン・エネルギー
- スペシャル技・シールド2枚
- タイプ相性、STAB、シャドウ補正
- 交代と45秒クールタイム
- CMP（攻撃実数値優先）
- 能力変化と確率効果
- 100試合の勝率分析

## 現在の制限
- 12種の練習用プリセットのみ
- ランク1相当の固定ステータス。デカヌチャンとベロベルトは近似値を含む
- AIはヒューリスティックで、人間の高度な読みを完全再現しない
- 通信遅延や不具合は再現しない
- ターン境界の一部はコミュニティ再現モデル

GitHub Pagesへアップロード後、更新が反映されない場合はSafariのWebサイトデータまたはホーム画面アイコンを一度削除して追加し直してください。
