# PJCS Training Lab v0.5

Pokémon GO スーパーリーグ向けの6→3選出・自動対戦シミュレーターです。

## v0.5の主な修正

- 通常技81種のターン数を全件検査し、ログを技ごとの実ターン・実時間で表示
- ゲージ技240種は「固有ターン数なし」として処理し、発動中にバトル時計を進めない
- 技321種の表示名を日本語へ統一
- オニドリルを追加（合計321体）
- 順位表示を「PvPoke総合順位」と明記し、使用率と混同しないよう修正
- 2026年7月の公開環境記事で言及されたポケモンに「環境注目」タグを追加
- おすすめ選出に、0・1・2枚の同数シールド1対1検証による「誰が誰へ刺さるか」の6体別相性表を追加
- iPhoneで検索欄を押した際の自動ズームを抑止（入力文字16px以上・ダイアログ表示時の自動フォーカスを廃止）
- 内部の `(normal)` 表示を削除
- 全体UIをモバイル向けに刷新

## 注意

PvPoke順位はシミュレーション順位であり、実測使用率ではありません。順位を確認できた収録ポケモンだけに番号を表示します。使用率の公開された完全データがないため、アプリ内で架空の使用率％は表示しません。
