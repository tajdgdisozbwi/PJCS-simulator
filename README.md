# PJCS Training Lab v0.10.5

Pokémon GO スーパーリーグ向けの6→3選出・自動対戦シミュレーターです。

## v0.6の主な修正

- 「勝てる選出」の％を透明化
  - 相手の60選出をすべて同じ重みで評価
  - 上位候補は各選出につき60選出 × 3乱数 = 180試合
  - 勝数・敗数・総試合数をカード内に表示
  - 実際の大会使用率による重み付けは行っていないことを明記
- 選出解説を具体化
  - 選出した各ポケモンが、相手の誰に刺さるかを個別表示
  - この3体では回答できない「重い相手」を表示
  - 回答が1体に集中する相手、先発が全9シールド条件で明確に不利な相手を警告
  - 相手6体それぞれへの最善回答と0〜2枚×0〜2枚の全9シールド結果を表示
- 6体登録画面
  - 「変更」ボタンを必ず横書きに固定
  - A / D を廃止し、「攻撃」「防御」「HP」と日本語で表示
  - ポケモン名の横にスプライト画像を表示
- 選出画面・検索画面・データ一覧にも画像を追加
- v0.5以前の古いおすすめ結果を再表示しないようキャッシュを更新

## 勝率の意味

「勝てる選出」に出る％は、現実の大会勝率や使用率を予測した数字ではありません。相手の6体から作れる60通りを等確率と仮定し、シミュレーター内AIで自動対戦した結果です。

## ポケモン画像

画像ファイルはZIP内へ同梱していません。PokéAPIの公開スプライトリポジトリから外部読み込みし、通信できない場合は文字アイコンへ切り替わります。Pokémonおよび各画像の権利は各権利者に帰属します。


- v0.6.1: フォルム違いポケモンの画像表示を修正。原種とリージョンフォームが同じ写真にならないよう、対応済みフォームは個別スプライトを使用し、未検証の特別フォームは誤表示を避けるため文字アイコンへフォールバックします。


## v0.7 相手選出AI

- 相手の3体を手動で選ぶ画面を廃止しました。
- 相手AIは、あなたの公開6体と技・個体値だけを見て、相手側60通りを比較します。
- あなたが現在選んでいる3体は相手AIの入力に使いません。
- 36対面を0〜2枚×0〜2枚の全9シールド条件で検証し、通常技、ゲージ技、技回転、耐性、CMP、シャドウ補正を選出評価に使用します。
- 上位6候補を、あなたの60選出×2乱数（各120試合）で再検証します。
- 選出確定後に、先発・引き先候補・締め役の理由、刺さる相手、重い相手、次点候補を表示します。
- 相手の選出はあなたの確定まで非公開です。


## v0.7.1 修正
- 3体選出の6枚を横スクロール可能に変更
- 選出画面に相手の公開6体を常時表示
- 「1試合を自動対戦」を押すたびに試合番号と乱数シードを進め、次の対戦を実行
- 相手チームと同じポケモンを登録可能であることを選択画面に明記（同じチーム内の重複のみ不可）


## v0.9 修正
- 3体選出画面は、あなた側・相手側とも6体を「3行×2列」の配置に戻し、そのボード全体を横スワイプ可能に変更
- 1対1の刺さり判定を、同数シールド3条件だけでなく、こちら0〜2枚 × 相手0〜2枚の全9条件で検証
- 「刺さる」は全9条件で6勝以上かつ同数シールドで2勝以上を基準とし、シールド有利時だけ勝つ対面を明確な有利扱いしない
- 対面表示に全9条件の勝敗、同数シールド勝敗、3×3のシールド結果を表示
- 相手AIの先発・引き先・締め役評価も全9シールド条件へ更新
