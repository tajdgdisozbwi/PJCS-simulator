# PJCS Pokémon GO Simulator v0.4

iPhoneのSafariからホーム画面に追加して使える、Pokémon GOスーパーリーグ／PJCS向けのPWAです。

## v0.4の主な修正

- **320体をアプリ内に同梱**し、外部サイトの読み込みに依存しない構成へ変更
- 通常・フォルム269体、シャドウ51体を収録
- 通常技81種、ゲージ技240種を同梱
- 各ポケモンについて、CP1500以下のSCP最大個体を4096通りのIVから計算
- レベル、攻撃IV、防御IV、HP IV、CP、攻撃・防御・HP実数値を表示
- 「耐久・SCPランク1」「攻撃寄り」「CMP寄り」「手入力」を選択可能
- 合法な通常技・ゲージ技から技構成を変更可能
- 収録一覧、検索、データ品質、除外理由をアプリ内で確認可能
- 3対3自動対戦、100試合勝率分析、60通りの選出アドバイザーを維持
- エンペルトの通常技名を **「きんぞくおん」** に統一

## データについて

内蔵データは、Pokémon GOマスターファイル型データの固定スナップショットを基に生成しています。
基準日はアプリの「データ」画面に表示されます。

収録順は公式使用率ではありません。競技環境で利用しやすいポケモンを前方に置いた収録順であり、勝率・おすすめ選出は本アプリのAIモデルによる推定値です。
